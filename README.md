# rn-skia-progress

React Native library which allows the use of high quality progress components with Skia

## Installation

```sh
npm install rn-skia-progress
```

## Usage

```js
import { multiply } from 'rn-skia-progress';

// ...

const result = await multiply(3, 7);
```
